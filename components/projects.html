Development Side Quests

What good is designing something if you can’t build it? Here are some simple web apps I’ve made to learn, experiment, and explore.

Fair Share Calculator

This project helps couples with different incomes split expenses equitably. By entering both partners' after-tax salaries and shared expenses, the tool calculates a fair division—no guesswork needed.

kaomoji.click

This project allows anyone to easily find, click, and copy kaomojis (Japanese emoticons) directly to their clipboard. I aimed to make the design as language-free as possible so it’s usable by the widest audience. (⁀ᗢ⁀)

Lost Cities Score Calculator

This one’s pretty niche! It’s a tool to calculate your score at the end of a game of Lost Cities. The design is unconventional but intentionally skeuomorphic, mimicking the way the game is played.
